## <a name="introduction">ü§ñ Introduction</a>

Metaversus - Modern UI/UX website, developed using Next.js, Framer Motion and Tailwind CSS, exemplifies modern UI/UX principles. Its sleek design, seamless animations, and overall user experience set a high standard, serving as a reference or inspiration for future modern applications or websites in general.

## <a name="tech-stack">‚öôÔ∏è Tech Stack</a>

- Next.js
- Framer Motion
- Tailwind CSS

## <a name="features">üîã Features</a>

üëâ **Beautiful Sections**: Includes hero, about, explore, features, get started, what's new, world, insights, feedback, footer, and navbar.

üëâ **Parallax Animations**: Engaging effects triggered by mouse movement and scrolling

üëâ **Latest UI Trends**: Incorporates modern design elements such as bento grids.

## <a name="quick-start">ü§∏ Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

*Make sure you have the following installed on your machine:*

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) *(Node Package Manager)*

**Cloning the Repository**

```bash
git clone https://github.com/andiondavi/metaversus.git
cd metaversus
```

**Installation**

*Install the project dependencies using npm:*

```bash
npm install
```

**Running the Project**

```bash
npm run dev
```

*Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.*
